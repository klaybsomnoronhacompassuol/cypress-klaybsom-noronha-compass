# 🚀 Cypress 

## 📚 Sobre o Projeto

Este repositório foi criado como parte da atividade prática do módulo de Automação de Testes Web com Cypress.

O objetivo desta atividade foi configurar o ambiente de automação e desenvolver o primeiro teste End-to-End (E2E) utilizando Cypress.

---

## 🛠 Tecnologias Utilizadas

- JavaScript
- Cypress
- Node.js
- Git & GitHub
- Visual Studio Code

---

## ⚙️ Configuração do Projeto

Para executar este projeto localmente:

### 1. Clone o repositório

```bash
git clone <url-do-repositorio>
```

### 2. Instale as dependências

```bash
npm install
```

### 3. Abra o Cypress

```bash
npx cypress open
```

---

## 🧪 Automação Web

O primeiro teste automatizado desenvolvido realiza as seguintes ações:

- Acessa uma página web
- Localiza elementos na tela
- Realiza interações utilizando clique
- Executa validações (assertions) para verificar o comportamento esperado da aplicação

O objetivo foi compreender o fluxo básico de automação End-to-End utilizando Cypress e praticar conceitos fundamentais de testes automatizados.

---

## 📁 Estrutura do Projeto

```bash
cypress/

cypress.config.js
package.json

```

---

## ▶️ Como Executar os Testes

### Modo Interface Gráfica

```bash
npx cypress open
```

### Modo Headless

```bash
npx cypress run
```
